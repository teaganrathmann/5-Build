# 5-Build
Module 5 Build
Indie Film Collective Website

## Project Description
This project is a multi-page website design. It showcases film screenings, workshops, and collaboration opportunities for independent filmmakers. The site was built using HTML and CSS with a mobile-first approach, and the layout adjusts for larger screens using responsive design techniques.

All HTML files are valid and properly structured, and the CSS is valid, fully functional, and applied across all pages consistently. This project does not include JavaScript, as it focuses on HTML and CSS only.

## Pages Included
Home (index.html)
Film Screenings (screenings.html)
Workshops (workshops.html)
Collaborations (collaboration.html)
